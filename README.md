# WeatherDash (Final Assignment)

A React + Vite weather web app using Open-Meteo APIs.

## Features
- Search city name
- 7-day forecast (max/min)
- Loading and error handling
- Day detail page (dynamic route)
- React Router multi-page structure

## Tech
- React (Vite)
- React Router DOM
- Fetch API
- Open-Meteo Geocoding + Forecast

## Run locally
```bash
npm install
npm run dev
